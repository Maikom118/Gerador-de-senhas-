# Password Lab

Aplicacao de geracao e avaliacao de senhas com:

- Script Python em `scanner.py`
- Front-end em `index.html`, `styles.css` e `app.js`

## Executar front-end

Abra `index.html` no navegador.

## Funcionalidades

- Gerar senha segura
- Digitar sua propria senha
- Avaliar forca da senha em tempo real
- Copiar senha
- Tema claro/escuro
