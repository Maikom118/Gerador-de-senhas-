# 🔐 Gerador de Senhas



> Um aplicativo intuitivo para criação de senhas seguras e aleatórias, permitindo ao usuário definir critérios específicos de segurança.

---

##  Funcionalidades

* **Comprimento Personalizável:** Defina a quantidade de caracteres da senha.
* **Parâmetros de Segurança:** Opções para incluir maiúsculas, minúsculas, números e caracteres especiais.
* **Copy to Clipboard:** Copie a senha gerada com apenas um clique.
* **Interface Responsiva:** Design adaptável para desktop e dispositivos móveis.

## 🛠️Tecnologias Utilizadas

O projeto foi construído utilizando o tripé fundamental do desenvolvimento Web:

* **HTML5:** Estrutura semântica.
* **CSS3:** Estilização e layout moderno.
* **JavaScript (ES6+):** Lógica de geração aleatória e manipulação do DOM.
